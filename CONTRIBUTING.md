# Contributing to Movable Type Framework

Thank you for your interest in contributing to Movable Type Framework! We welcome contributions from everyone.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/mt-framework.git`
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Commit your changes: `git commit -am 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature`
7. Submit a pull request

## Code Standards

- Follow PSR-12 coding standards
- Write tests for new features
- Document your code with comments
- Use meaningful commit messages

## Testing

Run tests with:
```bash
composer test
```

## Reporting Issues

- Check existing issues first
- Provide a clear description
- Include steps to reproduce
- Add any relevant error messages or logs

Thank you for contributing!
